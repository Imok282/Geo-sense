## Getting Started

### Prerequisites
- Node.js 18+
- Chrome/Edge browser (for Web Bluetooth)
- ESP32 or Arduino board

### Installation

```bash
# Clone the repository
git clone https://github.com/your-repo/geosense-pro.git
cd geosense-pro

# Install dependencies
npm install

# Set up environment variables
cp .env.example .env.local
# Add your GEMINI_API_KEY to .env.local

# Run development server
npm run dev
```

### Hardware Setup

1. **Water Node**: ESP32 + TDS Probe + HC-SR04 ultrasonic sensors
2. **Air Node**: ESP32 + MQ-135 + DHT22
3. **Climate Node**: ESP32 + DHT22 + BMP280 + BH1750 + Rain Sensor
4. **Seismic Node**: ESP32 + MPU6050 + ADXL345 + SW-420
